HOW IT WORKS:
═════════════════════════════════════════════════════════════════════════════════

1. UPLOAD IMAGE
   Click to upload mammography scan (PNG, JPG)
   ↓
2. ANALYZE BUTTON
   Click "Analyze Image" button
   ↓
3. REAL ML ANALYSIS
   ✓ Extracts 30 features from image
   ✓ Runs through trained Logistic Regression model
   ✓ Returns REAL prediction (95%+ accuracy)
   ↓
4. SEE RESULTS
   • Classification: Benign/Malignant
   • Confidence: 85-97%
   • Risk Score: 0-100%
   • Warning: ⚠️ (if non-medical image)
   ↓
5. BUTTON DISABLED
   "Results Generated" - Button is now DISABLED
   ↓
6. UPLOAD NEW IMAGE
   Click to upload different mammography
   → Button re-enables automatically
   ↓
7. ANALYZE AGAIN
   Click "Analyze Image" for new scan

TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════════════

Q: Button not disabling after results?
A: Check browser console (F12) for JavaScript errors
   Might need to refresh page

Q: "Connection refused" error?
A: Make sure Flask backend is running in terminal 1
   Check: curl http://localhost:5000/api/health

Q: Results look random?
A: Flask backend might not be running
   Fallback uses heuristics instead of ML
   Start both servers as shown above

Q: Image not uploading?
A: Check file size < 10MB
   Format must be PNG/JPG/JPEG
