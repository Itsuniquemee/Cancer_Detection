**Change Brand Color**
```html

```

**Modify Section Content**
```html
Find section by ID and edit text:
<section id="features">...</section>
<section id="analysis">...</section>
```

**Add New Feature Card**
```html
Copy existing .feature-card div and modify:
<div class="feature-card reveal">
    <div class="feature-icon">...</div>
    <h3>Your Feature</h3>
    <p>Your description</p>
</div>
```

**Adjust Animations**
```css
Find @keyframes section and modify:
@keyframes fadeInUp {
    transform: translateY(60px);  ← Change distance
}
```

## 🛠️ Troubleshooting

### **Server Won't Start**
```bash
# Check port 9000 is free
lsof -i :9000

# Use different port if needed
# Edit simple_server.py: server_address = ('127.0.0.1', 9001)
```

### **Styles Not Loading**
- Clear browser cache (Ctrl+Shift+Del)
- Check browser console (F12)
- CSS is embedded - should always work

### **Animations Laggy**
- Disable browser extensions
- Update GPU drivers
- Check system resources
- Use latest Chrome/Firefox

### **Mobile Not Responsive**
- Viewport meta tag is present
- Responsive breakpoints at 768px
- Zoom should work normally
