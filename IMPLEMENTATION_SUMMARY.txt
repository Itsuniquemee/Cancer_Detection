HOW IT WORKS:
═════════════════════════════════════════════════════════════════════════════════

User Flow:
  1. Visit http://localhost:8000
  2. Upload mammography image
  3. Click "Analyze Image" (button ENABLED)
  4. System:
     a. Converts image to grayscale
     b. Extracts 30 features (edge, texture, histogram)
     c. Sends to Flask backend
     d. ML model predicts: Benign or Malignant
     e. Returns confidence & risk score
  5. See results (button now DISABLED)
  6. Upload new image (button RE-ENABLED)
  7. Analyze again with new results

Feature Extraction (30 Features):
  1-5:   Basic statistics (mean, std, max, min, var)
  6-10:  Edge detection (Canny algorithm)
  11-20: Histogram analysis
  21-30: Texture percentiles

Prediction:
  Features → StandardScaler → Logistic Regression → P(Malignant)
  
  High probability (>0.5) → "Malignant"
  Low probability (<0.5) → "Benign"
  
  Risk Score = P(Malignant) × 100

INTEGRATING YOUR MODEL:
═════════════════════════════════════════════════════════════════════════════════

If you have your own trained Logistic Regression model:

1. Export your model:
   >>> import joblib
   >>> joblib.dump(your_model, 'breast_cancer_model.pkl')
   >>> joblib.dump(your_scaler, 'scaler.pkl')

2. Place files in project root:
   /Users/manas/Maanas/BreastCancerDetectionWeb/
   ├── breast_cancer_model.pkl
   └── scaler.pkl

3. Restart Flask:
   $ python3 app.py
   
   It will auto-load your model:
   model = joblib.load('breast_cancer_model.pkl')
   scaler = joblib.load('scaler.pkl')

4. Your model is now used for all predictions!
