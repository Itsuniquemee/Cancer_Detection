🔄 DATA FLOW:
═════════════════════════════════════════════════════════════════════════════════

User Interaction:
  1. Open http://localhost:8000
  2. Upload PNG/JPG image (<10MB)
  3. Click "Analyze Image" button (ENABLED)
  4. ↓

Image Processing:
  5. Convert to base64
  6. Send to backend via POST /api/predict
  7. ↓

Backend Analysis:
  8. Decode base64 → PIL Image
  9. Convert to grayscale
  10. Resize to 256×256
  11. Extract 30 features:
      - Basic stats (5): mean, std, max, min, var
      - Edge detection (5): Canny algorithm
      - Histogram (10): 256 bins
      - Texture (10): percentiles
  12. Normalize with StandardScaler
  13. Run Logistic Regression predict
  14. Get probability & confidence
  15. ↓

Results Display:
  16. Show classification (Benign/Malignant)
  17. Show confidence (85-97%)
  18. Show risk score (0-100%)
  19. Show warning (if non-medical image)
  20. Disable button ("Results Generated")
  21. ↓

User Can:
  22. Download report
  23. Upload new image (button RE-ENABLES)
  24. Analyze again
