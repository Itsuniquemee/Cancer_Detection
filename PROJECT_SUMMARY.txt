HOW TO USE
════════════════════════════════════════════════════════════════════════════════

OPTION 1: Frontend Only (No Backend)
  1. Open index.html in browser (or use Python server)
  2. Upload image and click "Analyze Image"
  3. See simulated results (no real ML needed)

  Command:
  $ python3 -m http.server 8000
  # Open: http://localhost:8000

OPTION 2: Full Stack with Backend
  1. Install dependencies: pip install -r requirements.txt
  2. Start Flask server: python app.py
  3. Open http://localhost:5000 in browser
  4. Upload image and click "Analyze Image"
  5. See REAL ML predictions from your model

  Commands:
  $ python3 -m venv venv
  $ source venv/bin/activate
  $ pip install -r requirements.txt
  $ python app.py
  # Open: http://localhost:5000

OPTION 3: Docker Deployment
  $ docker build -t cancerdetect .
  $ docker run -p 5000:5000 cancerdetect
  # Open: http://localhost:5000

INTEGRATING YOUR ML MODEL
════════════════════════════════════════════════════════════════════════════════

Your Jupyter notebook is at:
  /Users/manas/Maanas/Breast_Cancer_Logistic_Regression.ipynb

To use your model:

1. Export the trained model from Jupyter:
   import joblib
   joblib.dump(your_model, 'breast_cancer_model.pkl')
   joblib.dump(your_scaler, 'scaler.pkl')

2. Copy files to project root:
   cp /path/to/breast_cancer_model.pkl BreastCancerDetectionWeb/
   cp /path/to/scaler.pkl BreastCancerDetectionWeb/

3. Restart backend:
   python app.py

4. Done! Your model is now serving predictions.

The system automatically loads .pkl files if they exist, otherwise creates a demo.

API DOCUMENTATION
════════════════════════════════════════════════════════════════════════════════

1. Health Check
   GET /api/health
   Response: { status, version, model, timestamp }

2. Single Prediction
   POST /api/predict
   Body: { 
     "image": "data:image/png;base64,..." OR
     "features": [1.2, 3.4, 2.1, ...] (30 numbers)
   }
   Response: {
     "classification": "Benign|Malignant",
     "confidence": 95.2,
     "risk_score": 35.8,
     "malignant_probability": 0.358,
     "benign_probability": 0.642,
     "recommendation": { "urgency", "action", "next_steps" }
   }

3. Batch Prediction
   POST /api/batch-predict
   Body: {
     "cases": [
       { "id": "case_1", "features": [...] },
       { "id": "case_2", "features": [...] }
     ]
   }
   Response: {
     "results": [...],
     "total_cases": 2,
     "malignant_count": 1,
     "benign_count": 1
   }

4. Model Info
   GET /api/model-info
   Response: { model_type, accuracy, sensitivity, specificity, features }

5. Generate Report
   POST /api/generate-report
   Body: { "classification", "confidence", "risk_score" }
   Response: { "title", "results", "interpretation", "recommendations", "disclaimer" }
