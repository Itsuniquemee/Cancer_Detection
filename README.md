# Cancer Detection Web

Cancer Detection Web is a web application for breast cancer risk prediction and analysis workflows. It focuses on practical ML-assisted diagnosis support with a simple web interface.

## Tech Stack
- Python
- Flask
- HTML/CSS/JavaScript
- Scikit-learn and data science tooling

## Features
- Upload and analyze diagnostic data
- Prediction and probability reporting
- User-friendly web interface for clinical-style workflows
- Development and deployment guides included in repo

## How to Run
1. Clone the repo
   git clone https://github.com/Itsuniquemee/Cancer_Detection
2. Enter project folder
   cd Cancer_Detection
3. Install dependencies
   pip install -r requirements.txt
4. Start the app
   python app.py
5. Open browser at local host URL printed in terminal
